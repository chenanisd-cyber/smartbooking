[3] API REST PUBLIQUE AVEC HATEOAS
    - Ajouter spring-boot-starter-hateoas dans pom.xml
    - Créer package api.controller (séparé des controllers web)
    - GET /api/public/shows        → CollectionModel<ShowModel>
    - GET /api/public/shows/{id}   → EntityModel<ShowModel>
    - GET /api/public/artists      → CollectionModel<ArtistModel>
    - GET /api/public/artists/{id} → EntityModel<ArtistModel>
    - POST/PUT/DELETE /api/admin/* → ADMIN only
    - HTTP Basic auth pour l'API
    - CSRF désactivé pour /api/**
